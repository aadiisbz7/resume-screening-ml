# *BASIC* AI Resume Screening System (Machine Learning)

This project is a simple Machine Learning-based resume screening system.
It uses TF-IDF Vectorization and Cosine Similarity to match resumes with a job description.

## Features
- Takes job description as input
- Screens multiple resumes
- Ranks resumes based on matching score
- Uses NLP techniques

## Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python resume_screening_ml.py

## Example Output
- Resume ranking
- Matching percentage for each resume
